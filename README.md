# 📊 E-Commerce Product Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes an e-commerce product dataset using Microsoft Excel. The dataset was cleaned, analyzed, and visualized to generate meaningful business insights through Pivot Tables, charts, and an interactive dashboard.

## 🎯 Objectives
- Clean and prepare the dataset.
- Handle missing values.
- Apply Excel formulas.
- Perform descriptive statistical analysis.
- Create Pivot Tables and Pivot Charts.
- Build an interactive Excel dashboard.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis ToolPak
- VLOOKUP
- Slicers

## 📂 Dataset
The dataset contains approximately 100 product records with the following fields:
- Product ID
- Product Name
- Category
- Sub-Category
- Brand
- Cost
- Stock

## 🔄 Data Cleaning
- Converted data into an Excel Table.
- Removed duplicate records.
- Filled missing Cost values using the Average.
- Filled missing Stock values using the Median.
- Standardized Product ID formatting.

## 📈 Excel Functions Used
- AVERAGE()
- SUM()
- MAX()
- MIN()
- COUNTA()
- IF()
- RIGHT()
- VLOOKUP()

## 📊 Analysis Performed
- Descriptive Statistics (Mean, Median, Mode, Standard Deviation, Variance)
- Pivot Tables
- Pivot Charts
- Category-wise Analysis
- Stock Analysis
- Cost Analysis

## 📉 Dashboard Features
- Total Products KPI
- Average Product Cost KPI
- Total Stock Available KPI
- Highest Product Cost KPI
- Interactive Slicers
- Multiple Chart Types
  - Column Chart
  - Bar Chart
  - Pie Chart
  - Line Chart

## 📌 Key Insights
- Product distribution across different categories.
- Average product cost by category.
- Total stock available by category and sub-category.
- Identification of highest-cost products.
- Interactive filtering using slicers.
